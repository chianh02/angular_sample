luong khi commit (bat buoc phai pull truoc khi add, vi kho pull ma add thi van commit duoc)
pull => add => commit => push
neu bi config thi pull lay lai data da bi confilic(da commit) => fix lai => add roi commit nhu thương 

$ git add .(thêm file hoặc update file , truong hop moi merger xong(lay thay doi tu server roi add update cuar minh vao) thi phai chay
 add truoc roi moi commit duoc)
Sau lệnh add, bạn cần sử dụng câu lệnh Commit để đây thông tin thay đổi lên Local Respository:

$ git commit -m "Message"

$ git push origin <name_branch>
Ngoài ra, nếu chưa tồn tại remote trên server thì bạn cần phải add mới một remote trước rồi mới push:

$ git remote add origin <remote_url>
$ git push origin <name_branch>
